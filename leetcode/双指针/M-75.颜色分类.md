### 75. 颜色分类

给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数0、1和2分别表示红色、白色和蓝色。

注意:
不能使用代码库中的排序函数来解决这道题。

示例:

``` text
input: [2,0,2,1,1,0]
output: [0,0,1,1,2,2]
```

优化：

+ 一趟扫描且空间复杂度为O(1)。
